# Фотографии для секций главной страницы

Файлы из этой папки подхватываются автоматически по имени (расширение — webp/jpg/jpeg/png,
предпочтительно webp, горизонтальные ~1600×1200):

| Имя файла | Что на фото | Где используется |
|---|---|---|
| `demo-ha-cover.*` | Скрин дашборда HA (~1600×1000) | Секция «Живое демо» (кликабельная картинка) |
| `control-h2.*` | Aqara H2 на стене крупным планом | «Управление на любой вкус» |
| `control-relay.*` | Традиционный выключатель + реле в подрозетнике | «Управление на любой вкус» |
| `control-panel.*` | Панель Aqara S1E с включённым экраном | «Управление на любой вкус» |
| `control-tablet.*` | Планшет с интерфейсом умного дома | «Управление на любой вкус» |

Пока файла нет — на сайте отображается плейсхолдер с подсказкой.
URL демо-инстанса HA вписывается в `src/pages/index.astro` (константа `DEMO_HA_URL`).
